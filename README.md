# Rohit Selenium SignUp Test

Yeh project GitHub Actions ke saath Selenium automation test run karta hai.

## Kya karta hai?
- rahulshettyacademy.com/locatorspractice/ par jaata hai
- Username aur password enter karta hai
- "Forgot your password?" click karta hai
- Naye form mein Name, Email aur Phone fill karta hai

## GitHub par kaise push karein?

### Pehli baar:
```bash
git init
git add .
git commit -m "Selenium signup test"
git remote add origin https://github.com/TERA-USERNAME/TERA-REPO-NAME.git
git push -u origin main
```

### Dobara push karna ho toh:
```bash
git add .
git commit -m "Changes kiye"
git push
```

## Result kahan dekhein?
GitHub → Actions tab → Latest workflow run
